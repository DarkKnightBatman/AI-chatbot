# 🤖 LLM-Based Chatbot using LangChain, LangGraph & TinyLlama

This project is an AI-powered chatbot built using Hugging Face's TinyLlama model, integrated with LangChain and LangGraph to create a structured, stateful conversational system.

---

## 🚀 Features

- 🤖 Lightweight LLM (TinyLlama) for fast text generation  
- 🔗 Built using LangChain for LLM orchestration  
- 🌐 LangGraph-based workflow for structured conversation flow  
- 🧠 Memory-enabled chatbot for context-aware responses  
- ⚡ Efficient local inference using Hugging Face pipeline  

---

## 🛠️ Tech Stack

- Python  
- Hugging Face Transformers  
- LangChain  
- LangGraph  
- TinyLlama (1.1B Chat Model)
